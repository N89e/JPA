# Contributing

Merci de vouloir contribuer au projet Portfolio! 

## Comment contribuer

1. **Fork le repository**
```bash
git clone https://github.com/your-username/portfolio.git
```

2. **Créer une branche**
```bash
git checkout -b feature/votre-feature
```

3. **Faire vos changements**
```bash
# Éditer les fichiers
# Tester localement
```

4. **Commit vos changements**
```bash
git commit -m "Add: description de votre feature"
```

Messages recommandés:
- `Add:` pour une nouvelle fonctionnalité
- `Fix:` pour un bug fix
- `Refactor:` pour une restructuration
- `Docs:` pour la documentation
- `Style:` pour les changements de style
- `Test:` pour les tests

5. **Push vers votre fork**
```bash
git push origin feature/votre-feature
```

6. **Créer une Pull Request**
   - Description claire
   - Reference les issues associées
   - Screenshots si applicable

## Standards de Code

### Backend
- Utiliser ES6 modules
- Ajouter des commentaires pour le code complexe
- Respecter les standards Express.js
- Valider les inputs

### Frontend
- HTML5 sémantique
- CSS organise et responsive
- JavaScript vanilla ou ES6+
- Pas de dépendances externes inutiles

## Tests

Avant de soumettre votre PR:
```bash
cd backend
npm run dev

cd frontend
npm start
```

Tester manuellement toutes les features.

## Documentation

Si vous ajoutez une feature:
- Mettre à jour README.md
- Documenter les API endpoints
- Ajouter des exemples

## Questions?

Ouvrez une issue ou contactez: votre.email@example.com
