🔗 ENDPOINTS API
═══════════════════════════════════════════════════════════════════════════════════

GET  /api/health                    → Vérifier la connexion
GET  /api/services                  → Tous les services
GET  /api/projects                  → Tous les projets
GET  /api/projects/:id              → Un projet spécifique
POST /api/contact/submit            → Soumettre formulaire contact
